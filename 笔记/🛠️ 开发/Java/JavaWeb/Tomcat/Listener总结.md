# Listener总结

## 1.简介

Listener是一个实现了特定接口的Java程序用于监听一个方法或属性，当被监听的方法被调用或属性改变时会自动执行某个方法，也就是说：监听器就是监听某个域对象的状态变化的组件。

## 2.监听器

### 2.1 重要概念

- **事件(响应行为)**
  - 某个方法被调用或者属性改变
- **事件源**
  - 被监听的对象(ServletContext、request、session、...)
- **监听器**
  - 用于监听事件源当发生事件时会触发监听器。

### 2.2 监听器分类

- **按监听对象分类**
  - `ServletRequest域`
  - `HttpSession域`
  - `ServletContext域`
- **按监听内容分类**
  - 监听域对象创建与销毁
  - 监听域对象的属性变化
